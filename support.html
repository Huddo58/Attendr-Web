Getting Started

Most organisations can deploy Attendr in just a few minutes.

1. Download Attendr

Download the latest version from the download page and extract the package to your server or workstation.

2. Run Attendr

Launch the application and open Attendr in your browser using the local server address.

3. Create Admin Account

The first login creates the administrator account for your organisation.

4. Add Departments

Create departments that match your team structure.

5. Add Users

Add users and assign them to departments.

6. Start Using Attendr

Users can now update their status and location in real time.
